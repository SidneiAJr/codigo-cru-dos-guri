## Comandos para instalar o Typescript:

 - npm install typescript -D
 - npx tsc --init

## Configuração do Typescript:
 - module : "commonjs"
 - target : "es6"
 - rootDir : "./src"
 - outir : "./dist"
   
## Comando para iniciar aplicação em TYpescript
- npx ts-node nome_arquivo ou pasta/nome_arquivo
